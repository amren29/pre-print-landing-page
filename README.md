# Pre-Print Landing Page

Landing page for **Pre-Print** — an automated artwork preflight tool for print shops. Pre-Print validates customer-uploaded artwork files against print specifications before they go to press.

## What is Pre-Print?

Pre-Print helps print shops stop chasing customers over file issues. It runs instant preflight checks on size, bleed, DPI, color mode, and safe area so only production-ready artwork reaches the press.

## Page Sections

- **Features** — Product templates, multi-item orders, digital proofing, and more
- **Workflow** — Step-by-step accordion showing how the preflight process works
- **Process** — Getting started guide in four simple steps
- **Pricing** — Plans for print shops of all sizes

## Tech Stack

- Static HTML single-page site
- Tailwind CSS for styling
- Hotwired (Turbo + Stimulus) for interactivity
- Google Fonts (Inter, Outfit, Anek Telugu, Source Code Pro)

## Getting Started

Open `index.html` in a browser or serve it with any static file server:

```bash
npx serve .
```
